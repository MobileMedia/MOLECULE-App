5 Technologies

Following a technocentric definition of mobile learning most people think of mobile learning as using smartphones. In the mobile learning research though more and more all kinds of mobile and ubiquitous technologies become important.

In essence one can distinguish several technology categories that are important for building mobile media and learning support, these are:

  • Sensing technologies and sensors, one can differentiate classical hardware sensors and software sensors.
  • Mobile apps used on tablet and smartphone devices, mainly there are personal devices and personal services in a mobile learning context. Advanced mobile apps also use sensor technologies to be more context aware.
  • Notification systems are essential form synchronizing different parts of a ubiquitous learning environment and trigger context actions.
  • Ambient display technologies and augmented reality solutions.
  • Augmented Reality.
  • MashUp technologies, PLEs, and mixed reality solutions.
